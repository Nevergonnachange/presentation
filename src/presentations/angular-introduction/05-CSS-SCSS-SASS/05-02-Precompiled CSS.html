Precompiled CSS

“Da Sass ein Präprozessor für CSS ist, ersetzt es nicht die Notwendigkeit entsprechender CSS-Kenntnisse! Diese sind eine Grundvoraussetzung für die erfolgreiche Nutzung von Sass. Sollten Sie also noch nicht über ausreichende CSS-Kenntnisse verfügen, wird dringend empfohlen, sich diese zuerst anzueignen, bevor Sie sich mit Sass beschäftigen!”
